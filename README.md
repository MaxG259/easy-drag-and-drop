## Drag & Drop демо
Простой демонстрационный проект с нативным Drag & Drop на чистом JavaScript

### Демо
- **Live (Vercel)**: `https://easy-drag-and-drop.vercel.app/`

### Стек
- HTML5, CSS3
- Ванильный JavaScript (ES6+)

### Основные возможности
- Перетаскивание элементов мышью (Drag & Drop API)
- Подсветка зон при наведении
- Корректная обработка событий dragstart/dragover/drop
- Адаптивные стили
